# SQL Queries for Customer Churn Analysis

This folder contains all SQL scripts used to analyze the IBM Telco Customer Churn dataset.

## Files

1. **01_churn_rate.sql**  
   - Calculates the overall churn rate for all customers.
   - Outputs total customers, number of churned customers, and churn rate percentage.

2. **02_churn_by_contract.sql**  
   - Calculates churn rate grouped by contract type (Month-to-month, One year, Two year).
   - Outputs total customers, number of churned customers, and churn rate percentage per contract type.
     
3. **03_Churn_By_SeniorCitizen**
   - Calculates churn rate grouped by SeniorCitizen(Yes, No).
   - Outputs total customers, number of churned customers, and churn rate percentage depending on Senior citizen.
