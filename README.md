# Cutomer_Churn_Analysis
Customer churn analsysis using Pythn, SQL, and Power BI

#Tools Used:-

- SQL (SQLite)
- DB Browser For SQLite
- Github For Documentation
- Power BI for Dashboard

#Key Insights:-

- Customers on **month-to-month contracts** have the highest churn rate.
- **Short-tenure customers (0â€“12 months)** churn significantly more than long-term customers.
- **Senior citizens** show a higher churn tendency compared to non-senior customers.
- Customers with **fiber optic internet** have higher churn than DSL or no internet users.

These insights suggest that long-term contracts and early customer engagements are critical for reducing churn.
